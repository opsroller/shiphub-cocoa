4.1.0-beta.2

  • [BUGFIX] Fixes an issue where the whole app’s Application Support directory (sandboxed apps) / user’s Application Support directory (non-sandboxed apps) was accidentally excluded from backups. This SDK release explicitly includes the Application Support directory into backups. If you want to opt-out of this fix and keep the Application Directory’s backup flag untouched, add the following line above the SDK setup code:

  • Objective-C:

    [[NSUserDefaults standardUserDefaults] setBool:YES forKey:@"BITExcludeApplicationSupportFromBackup"];

  • Swift:

    NSUserDefaults.standardUserDefaults().setBool(true, forKey: "BITExcludeApplicationSupportFromBackup")
